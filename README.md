# Robot Market

Run Locally

1. Clone repo
    $ Clone repo $ git clone https://github.com/lexican/robot-market.git <br>
    $ cd robot-market
    $ npm install
    # npm run dev

2. Run Backend <br>
    $ cd server <br>
    $ npm install <br>
    $ npm start

3. Run Frontend <br>

    open new terminal <br>
    
    $ cd frontend <br>
    $ npm install<br>
    $ npm start
